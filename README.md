「Akamura」は福岡県田川郡赤村の公式観光アプリです．


## 機能
🌿村のイベントの一覧<Br>
🌿レビュー投稿・閲覧<Br>
🌿施設紹介<Br>
🌿観光マップ


## 開発について
このアプリは学部3年生の後期にプログラマ3人で開発しました．<Br>
設定，レビュー，施設紹介を担当しました．
